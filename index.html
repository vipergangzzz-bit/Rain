Install

  1. Install Forge 1.8.9 — Run the installer once, Make a Forge launcher profile
  2. Download the jar — Download rain AC.jar from the button above
  3. Copy to mods — Drop it in %APPDATA%\.minecraft\mods
  4. Join a server — Launch Forge 1.8.9 and play

Usage

Press RShift to open the GUI. Toggle checks, set flash and nametag colors. Flagged players show red in tab and in-world.

Features

Observer-side Reads position, rotation, and state of entities around you using only data your client already receives.
Evidence ledger Flags from different detection families corroborate. One-off blips decay; multiple families together trigger an alert.
Reflection fallback No Searge mappings? No problem. Works on vanilla or other Forge versions without crashing.
Flash + nametags Screen flash and chat on first detection. Flagged players get colored nametags in-world and in tab.
